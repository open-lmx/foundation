# Open-LMX Foundation

Open-LMX Foundation defines the open, vendor-neutral foundation for interoperable education, training, learning, skills, certification, credentials, employment, recruitment, and career progression.

LMX stands for **Learning Model eXchange**.

The education, employment, recruitment, and industry ecosystems lack a trusted common language. Educational institutions often teach content that does not map clearly to real work. Employers repeatedly find graduates difficult to evaluate or not ready for work. Learners struggle to prove what they know and what they can do. Recruiters and hiring managers lack portable, governed, evidence-based skill signals.

Open-LMX exists to reduce this friction.

It first defines a common operating model for the education and employment sector, then enables platforms and services to be built on top of that model.

## North Star

A trusted, open, interoperable language for learning, skills, credentials, work readiness, hiring, growth, and career progression.

## Core Belief

Learning must connect to skills.  
Skills must connect to work.  
Work must connect to evidence.  
Evidence must connect to credentials.  
Credentials must connect to trust.  
Trust must be portable across institutions, employers, platforms, and countries.

- Employment

## Core Questions

Open-LMX should help answer:

- What does a learner know?
- What can a learner do?
- How was that skill evaluated?
- Who verified it?
- What evidence exists?
- Is the credential trusted?
- Is the person ready for a role?
- Which skill gaps remain?
- Which learning path closes those gaps?
- Which jobs, projects, or careers does this enable?
