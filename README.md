# Open-LMX Foundation

Open-LMX Foundation defines the open, vendor-neutral foundation for governed, interoperable, AI-native systems.

LMX stands for **Language Model eXchange**.

The foundation provides shared principles, schemas, lifecycle models, governance theory, trust theory, and reference models for agents, models, tools, skills, prompts, memory, datasets, workflows, policies, evaluations, and AI-native applications.

## North Star

Any agent.  
Any model.  
Any tool.  
Any cloud.  
Any device.  
Any protocol.  
Governed by open standards.

## Core Belief

Everything is an asset.  
Every asset has identity.  
Every identity has lifecycle.  
Every lifecycle is governed.  
Every action is observable.  
Every decision is auditable.  
Every state is temporal.  
Every system must be interoperable.

## Scope

Open-LMX Foundation defines canonical models for:

- Assets
- Identity
- Relationships
- Lifecycle
- Governance
- Trust
- Policy
- Evaluation
- Certification
- Interoperability

## Status

This repository is the initial foundation layer. Specifications are draft until marked as approved.
