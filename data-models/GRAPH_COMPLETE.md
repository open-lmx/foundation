# Open-LMX Complete Graph (Canonical)

This document defines the complete unified graph model for Open-LMX Foundation.

It connects education, capability, work, employment, social validation, trust, and career progression into one interoperable system.

---

# 1. Core Unified Lifecycle

```text
Knowledge
  ↓
Learning
  ↓
Assessment
  ↓
Competency
  ↓
Capability
  ↓
Role
  ↓
Job / Work
  ↓
Employment
  ↓
Career
```

---

# 2. Supporting Graph Layers

## A. Social Graph (Validation Layer)

```text
Person ↔ Person
Person ↔ Institution
Person ↔ Employer
Person ↔ Mentor
Person ↔ Peer
```

Purpose:
- trust creation
- peer validation
- institutional endorsement

---

## B. Trust Graph (Inference Layer)

```text
Evidence → Signals → TrustScore → Decisions
```

Trust is computed from:
- social validation
- performance history
- assessment results
- credential issuance

---

## C. Capability Graph (Bridge Layer)

```text
Competency → Capability → Capability Requirement → Role
```

Purpose:
- convert learning into real-world ability
- match humans to work

---

## D. Work / Employment Graph (Demand Layer)

```text
Industry → Role → Job → Task → Performance
```

Purpose:
- define real-world demand
- structure execution and evaluation

---

## E. Learning Graph (Supply Layer)

```text
Knowledge → Concept → Objective → Lesson → Course → Program
```

Purpose:
- structure knowledge delivery
- generate measurable learning outcomes

---

## F. Career Graph (Temporal Layer)

```text
Job Experience → Performance → Role Progression → Career Path
```

Purpose:
- model human growth over time
- enable promotion and trajectory planning

---

# 3. Unified Multi-Graph System

Open-LMX is not a single graph.
It is a multi-layer graph system:

```text
                TRUST GRAPH
                    ↓
 SOCIAL GRAPH → CAPABILITY GRAPH → WORK GRAPH
                    ↓
             CAREER GRAPH (TIME)
                    ↑
           LEARNING GRAPH (SUPPLY)
```

---

# 4. Core Keystone Entities

The system depends on two critical bridges:

## Capability (Supply ↔ Demand bridge)
- transforms learning into work readiness

## Role (Demand anchor)
- defines structured real-world expectations
