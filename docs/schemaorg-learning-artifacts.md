# Schema.org Learning Artifacts

Open-LMX learning artifacts align with schema.org education and learning types.

## Core Schema.org Types

- LearningResource
- Course
- CourseInstance
- EducationalOccupationalCredential
- EducationalOccupationalProgram
- EducationEvent
- AlignmentObject
- DefinedTerm
- CreativeWork

## LearningResource

Schema.org defines LearningResource as a CreativeWork with an explicit orientation toward learning, education, skill acquisition, and educational purposes.

Important properties include:

- teaches
- assesses
- competencyRequired
- educationalAlignment
- educationalLevel
- educationalUse
- learningResourceType
- interactivityType
- isPartOf
- hasPart
- author
- publisher
- provider
- license
- inLanguage
- accessibilityFeature
- accessibilityHazard
- accessibilitySummary

## Mapping Rule

Every Open-LMX learning artifact should include schema.org alignment metadata where possible.

At minimum:

- schema_org_type
- schema_org_url
- teaches
- assesses
- competency_required
- educational_level
- educational_use
- learning_resource_type

## Governance Rule

Learning artifacts must declare what they teach, what they assess, their intended learner level, their source/provenance, and their evaluation criteria.
