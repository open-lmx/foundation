# Trust Theory

Trust is not assigned.

Trust is earned through evidence.

## Trust Inputs

- Identity verification
- Provenance
- Evaluation results
- Historical performance
- Reliability
- Security posture
- Compliance evidence
- Peer review
- User feedback
- Incident history

## Trust Rule

Trust must be explainable, reproducible, and reviewable.

## Contextual Trust

An asset may be trusted for one purpose and not trusted for another.

Trust is context dependent and should be evaluated against intended use.
