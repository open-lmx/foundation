# Evaluation Theory

Evaluation determines whether an asset is fit for a defined purpose.

## Evaluation Dimensions

- Capability
- Quality
- Reliability
- Security
- Compliance
- Safety
- Cost
- Performance
- Explainability
- User Experience

## Evaluation Requirements

Every evaluation should capture:

- Scope
- Method
- Inputs
- Outputs
- Evidence
- Metrics
- Reviewer
- Timestamp

## Evaluation Rule

Claims without evaluation evidence should be treated as unverified.
