# Relationship Theory

Assets do not exist in isolation.

Open-LMX models systems as networks of assets connected by typed relationships.

## Core Relationship Types

- USES
- OWNS
- CREATED_BY
- APPROVED_BY
- DEPENDS_ON
- GOVERNS
- IMPLEMENTS
- CERTIFIED_BY
- DEPLOYED_TO
- REPORTS_TO
- MEMBER_OF
- TRUSTED_BY
- DERIVED_FROM

## Relationship Requirements

Every relationship should declare:

- Source
- Target
- Type
- Creation timestamp
- Validity period

## Relationship Rule

Relationships are first-class assets and should be traceable through time.
